### 🎯 Основная гипотеза

**Гипотеза**: Многоклеточность в виде дифференцировки тканей у *Volvox carteri* эволюционировала ч
1) Через **перенос временных программ** *Chlamydomonas* в **пространственные программы** (2 типа клеток - germ \ soma).
2) Через **некодирующие белки регуляторные элементы** (промоторов, энхансеров, некодирующих\структурных РНК и тд),  **а не через эволюцию белок-кодирующих генов**.

### 🔍 Научный вопрос

Как (и какую роль в этом занимают некодирующие РНК) произошла эволюция многоклеточности и дифференциации тканей в линии зеленых водорослей?

---

## 🌟 Почему это интересно?

- **Простая модельная система**: *Volvox* и *Chlamydomonas* - одни из самых простых организмов для изучения эволюции многоклеточности
- **Независимая эволюция**: Многоклеточность у *Volvox* возникла **независимо** от животных/растений - возможна паралельная эволюция
- **Минимальная сложность**: Только 2 типа клеток (соматические и герминативные) → идеальная система для изучения 

### Эволюционное значение

- **Regulatory vs coding evolution**: Что важнее - новые гены или новая регуляция старых генов?
- **Temporal-to-spatial transition**: Как временные программы становятся пространственными?

### 📋 ПОДГОТОВИТЕЛЬНЫЕ ЭТАПЫ проекта:

1) разработать системы культивирования с фиксированием изменяемых переменных культуры (температура / ph / уровень CO2 / свет - с помощью Arduino UNO)
2) написать код на C для связи сенсоров (CO2, температуры, ph, освящения) через Arduino UNO и вывод показателей через WiFi для постоянного контроля условий
3) выделение 2ух ферментов Вольвокса разрушающих ECM - главная преграда для выделения ядер
4) перенос и адаптация протокола выделения интактных ядер с Хламидомонады на Вольвокс для последующего single nucleus RNA seq
